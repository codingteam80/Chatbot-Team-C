A. Requires download of Python 3.11.9:

Steps to download Python 3.11.9
1. Go to the official download page: https://www.python.org/downloads/release/python-3119/ (python.org in Bing)
2. Scroll down to Files and choose the installer that matches your operating system:
    For Windows → Windows installer (64-bit)
    For macOS → macOS 64-bit universal2 installer
    For Linux → usually installed via package manager, but you can grab the source tarball.


B. The chatbot also requires to download Ollama, model-qwen3:1.7b:

Steps to download Ollama 
1. Visit https://ollama.com/download
2. Download the installer for your OS (Windows, macOS, Linux)
3. Open Terminal and type: 
     ollama pull  qwen3:1.7b
